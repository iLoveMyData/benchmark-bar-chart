# Support

Thanks for using **Benchmark Bar Chart**! Here is how to get help.

## Found a bug? Have a feature request?

Please open an issue on GitHub:
https://github.com/iLoveMyData/benchmark-bar-chart/issues

When reporting a bug, including the following helps a lot:

- The version of the visual (visible in the format pane → About / or in the `.pbiviz` filename).
- The Power BI environment (Desktop / Service / mobile, host locale).
- A short description of what you expected vs. what happened.
- A screenshot or, if possible, a sanitized `.pbix` reproducing the issue.
- Browser console errors (open DevTools → Console) if you're on Power BI Service.
